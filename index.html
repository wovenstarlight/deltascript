A project aimed at creating a complete script of Deltarune by Toby Fox.

Inspired by the ISAT Script Project.

Site tips

The script has been organized by chapters and locations. (ISP has it like that, story (subdivided by locations) and locations (subdivided by story). Unless there's some more logical way of doing it...)

Check the bottom right menu for extra features:

  • Compact mode: Just the dialogue, no sprites, so you can focus on what's being said.
  • Dyslexia-friendly mode: Switches to the OpenDyslexic font for easier reading.
  • Show all choices: Displays results of all choices for easier searching.

Contribute

This is a community project, and we welcome help! Check out the Github page to see how you can contribute.
